# 🎬 Java 영화 예매 시스템 (Movie Reservation System)

## 📌 프로젝트 개요

본 프로젝트는 Java의 객체지향 프로그래밍(OOP) 핵심 문법(캡슐화, 상속, 다형성)과 자료구조(ArrayList), 예외 처리, 파일 입출력 등을 종합적으로 활용하여 구현한 콘솔 기반의 영화 예매 시스템입니다.
현실적인 영화관 비즈니스 로직(지점별 교차 상영, 장르 필터링, 세분화된 요금표, 특정 좌석 배치)을 완벽하게 콘솔에 구현하는 것을 목표로 하였습니다.

## 🚀 주요 기능 (Key Features)

### 1. 사용자 기능 (Customer)

**회원가입 및 로그인:** 아이디 중복 검사 및 객체지향적 회원 데이터 관리

- **지점 및 장르 기반 영화 조회:**
- **중계점**과 **노원점** 중 원하는 상영관 지점 선택
- 전체 영화 보기 또는 **장르별(공포/코미디) 필터링** 기능 제공

**영화 예매:**

- 관람 대상(성인/청소년/어린이/경로/장애인)에 따른 맞춤 요금 적용
- 직관적인 2D 배열 기반의 좌석표(A~E행, 1~5열) 출력 및 선택
- 예매 완료 시 결제 금액의 5% 멤버십 포인트 적립

**예매 관리:** 본인의 예매 내역 조회 및 예매 취소 기능 (취소 시 좌석 상태 원상복구)

### 2. 관리자 기능 (Admin)

**초기 데이터 자동 세팅:** 시스템 최초 실행 시 관리자 계정(`admin`/`1234`) 및 지점별 상영 스케줄 자동 생성

- 영화 정보 및 상영 스케줄(오전/오후, 지점별 상영관 할당) 관리
- 회원 목록 조회 및 시스템 전반의 예매 데이터 관리

## 💡 특화 비즈니스 로직

**① 지점별 교차 상영 로직**

- **중계점:** 오전 시간대에는 공포 영화(`군체`), 오후 시간대에는 코미디 영화(`와일드씽`) 상영
- **노원점:** 오전 시간대에는 코미디 영화(`와일드씽`), 오후 시간대에는 공포 영화(`군체`) 상영

**② 지정석(장애인석/프리미엄석) 배치 및 요금 할증**

- 상영관은 5x5 (총 25석) 규모로 생성됩니다.
- `[0][0]` 즉, **맨 앞줄 가장 좌측(A1)** 좌석은 **장애인 전용석**으로 고정 배치됩니다.
- 마지막 줄인 **E행** 전체는 **프리미엄석**으로 지정되며, 예매 시 2,000원이 할증됩니다.
- 오전 시간대 선택 시 조조할인이 적용됩니다.

## 🛠 적용된 Java 핵심 문법 (기본 구현 조건 충족)

1. **객체지향 및 캡슐화:** 모든 데이터 클래스(`Movie`, `Seat`, `Schedule` 등)의 필드는 `private`으로 선언하고 Getter/Setter를 통해서만 접근하도록 캡슐화 적용.
2. **상속 및 다형성:** 최상위 추상 클래스 `User`를 상속받아 `Customer`와 `Admin` 클래스를 구현.
3. **인터페이스 활용:** `PricingPolicy` 인터페이스를 선언하고, `StandardPricing` 클래스에서 이를 구현하여 요금 계산 로직을 유연하게 설계.
4. **ArrayList 사용:** 회원, 영화, 스케줄, 예매 내역 등 가변적인 데이터를 관리하기 위해 Java Collection Framework의 `ArrayList` 사용.
5. **사용자 정의 예외 처리:** `CinemaException` 클래스를 만들어 잘못된 좌석 번호, 이미 예매된 좌석, 없는 예매 내역 등 비즈니스 로직 오류를 `try-catch`로 안전하게 처리.
6. **파일 입출력 (데이터 영속성):** `Serializable` 인터페이스를 구현하고 `ObjectInputStream/ObjectOutputStream`을 사용하여 프로그램 종료 후에도 `cinema_final.dat` 파일에 데이터를 직렬화하여 영구 보존.

## 📁 파일 구조 및 클래스 설명

| 클래스명                   | 역할 및 기능                                                                        |
| -------------------------- | ----------------------------------------------------------------------------------- |
| **`MovieSystemMain.java`** | 시스템의 메인 실행 클래스 (메뉴 루프, 파일 I/O, 전체 프로그램 흐름 제어)            |
| **`CinemaData.java`**      | DB 역할을 하는 래퍼(Wrapper) 클래스 (ArrayList 데이터 모음)                         |
| **`User.java` (추상)**     | 회원 및 관리자의 공통 속성을 정의하는 최상위 클래스                                 |
| **`Customer.java`**        | 일반 사용자 클래스 (포인트 필드 포함, `User` 상속)                                  |
| **`Admin.java`**           | 시스템 관리자 클래스 (`User` 상속)                                                  |
| **`Movie.java`**           | 영화 정보 (제목, 장르) DTO                                                          |
| **`Schedule.java`**        | 상영 일정 (영화, 지점, 시간대, 해당 타임의 상영관 객체 포함)                        |
| **`Theater.java`**         | 상영관 내 5x5 좌석 2D 배열 관리 (장애인석, 프리미엄석 할당 및 예외 처리)            |
| **`Seat.java`**            | 개별 좌석의 상태 정보 (좌석번호, 예매여부, 장애인석/프리미엄석 여부)                |
| **`PricingPolicy.java`**   | 요금 계산을 위한 인터페이스                                                         |
| **`StandardPricing.java`** | 연령, 시간대, 좌석 등급을 고려하여 실질적인 결제 요금을 계산 (`PricingPolicy` 구현) |
| **`Reservation.java`**     | 최종 예매 정보 데이터 클래스                                                        |
| **`CinemaException.java`** | 시스템 전용 커스텀 예외 처리 클래스                                                 |

## ▶️ 실행 방법

1. JDK(Java Development Kit) 8 버전 이상이 설치되어 있는지 확인합니다.
2. 모든 `.java` 파일을 동일한 디렉토리(폴더)에 위치시킵니다.
3. 터미널(또는 명령 프롬프트)을 열고 해당 폴더로 이동한 후 아래 명령어로 컴파일합니다.

```bash
javac -encoding UTF-8 *.java

```

4. 컴파일이 완료되면 다음 명령어로 프로그램을 실행합니다.

```bash
java MovieSystemMain

```

5. 최초 실행 시 `admin` / `1234` 계정으로 로그인하여 전체 데이터를 확인하거나, 2번 `회원가입` 메뉴를 통해 새로운 사용자를 생성하여 예매 테스트를 진행할 수 있습니다.
