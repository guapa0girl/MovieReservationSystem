# 🎬 Java 영화 예매 시스템 (Movie Reservation System)

## 📌 프로젝트 개요

본 프로젝트는 Java 객체지향 프로그래밍(OOP)을 활용한 통합 영화 예매 시스템입니다. **다인원 동시 예매**, **지점별/장르별 필터링**, **포인트 적립 및 자동 회수**, **관리자용 데이터 초기화** 등 실무 비즈니스 로직을 포함하고 있습니다.

## 🚀 주요 기능 (Key Features)

### 1. 사용자 기능 (Customer)

- **회원가입 및 로그인:** 아이디 중복 검사 및 계정 관리.
- **상영 지점/장르 필터링:** 중계점/노원점 선택 후, 장르별(공포/코미디) 상영 스케줄만 필터링 조회.
- **다인원 동시 예매:** 최대 5명까지 인원을 선택하고, 각 인원별 연령대 입력 및 좌석을 연속 선택.
- **좌석 배치 로직:** 0,0 좌표(A1)는 **장애인석(W)**, E행은 프리미엄석(P)으로 자동 할당 및 관리.
- **포인트 시스템:** 결제 금액의 5% 적립. 예매 취소 시 적립된 포인트를 자동으로 차감(회수)하여 데이터 정합성 유지.

### 2. 관리자 기능 (Admin)

- **초기 자동 세팅:** 최초 실행 시 지점별 교차 상영 스케줄 및 관리자 계정 자동 생성.
- **회원 관리:** 일반 회원들의 누적 멤버십 포인트 실시간 조회.
- **데이터 강제 초기화 (RESET):** 시스템 전체 데이터를 초기 상태로 리셋하는 기능 제공.

## 🛠 적용된 Java 핵심 문법

- **OOP 4대 원칙:** \* **캡슐화:** `private` 필드와 `public` 메서드(Getter/Setter, 비즈니스 로직)를 통한 정보 은닉.
- **상속:** `User` 추상 클래스를 상속받아 `Customer`, `Admin` 구현.
- **다형성:** `PricingPolicy` 인터페이스와 오버라이딩을 활용하여 요금 계산 로직의 유연성 확보.
- **추상화:** 관리자/사용자 권한을 다형성 기반으로 처리.

- **자료구조:** `ArrayList`와 `배열(Array)`을 조합하여 다인원 예매 데이터 및 좌석 정보 관리.
- **데이터 영속성:** `Serializable` 인터페이스와 파일 직렬화를 통해 프로그램 종료 후에도 데이터 유지.
- **예외 처리:** `CinemaException` 커스텀 예외 클래스를 활용하여 좌석 예매 및 취소 시의 오류 제어.

## 📁 주요 클래스 설명

- **`MovieSystemMain.java`**: 컨트롤러 역할 (메인 메뉴 루프, 기능 통합)
- **`Theater.java`**: 5x5 좌석 배열(2D) 관리 및 좌석 특성 할당 (장애인석/프리미엄석)
- **`PricingPolicy.java`**: 요금 정책 추상 인터페이스
- **`Reservation.java`**: 개별 예매 내역 관리 (취소 시 금액 반환 로직 포함)
- **`CinemaData.java`**: 데이터 저장을 위한 Wrapper 객체 (ArrayLists 모음)

## ▶️ 실행 방법

1. JDK 설치 확인 후 모든 파일을 동일 폴더에 저장합니다.
2. 터미널에서 컴파일합니다:

```bash
javac -encoding UTF-8 *.java

```

3. 프로그램을 실행합니다:

```bash
java MovieSystemMain

```

4. 최초 로그인: 아이디 `admin`, 비밀번호 `1234`로 관리자 접속 가능.
